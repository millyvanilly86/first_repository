Im vorherigen Video hast du etwas über Git und GitHub gelernt. Bevor Sie mit diesem Video fortfahren, registrieren Sie sich für ein GitHub-Konto und melden Sie sich an. Beginnen wir mit der Erstellung eines neuen Repositorys. Klicken Sie auf + und dann auf Neues Repository. Um ein neues Repository zu erstellen, müssen Sie folgende Details angeben: Geben Sie Ihrem neuen Repository einen Namen, fügen Sie optional eine Beschreibung Ihres Repositorys hinzu, wählen Sie die Sichtbarkeit des Repositorys — ob es öffentlich oder privat sein soll — und wählen Sie die Option, dieses Repository mit einer Readme-Datei zu initialisieren. Klicken Sie anschließend auf Repository erstellen. Sie werden nun zu dem Repository weitergeleitet, das Sie erstellt haben. Der Stammordner Ihres Repositorys ist standardmäßig aufgeführt und enthält nur eine Datei README.md. Jetzt ist es an der Zeit, die Readme-Datei zu bearbeiten. Sie können dies in Ihrem Browser tun. Klicken Sie einfach auf den Stift, um den Online-Editor zu öffnen, und Sie können den Text der Readme-Datei ändern. Um Ihre Änderungen im Repository zu speichern, müssen Sie sie übernehmen. Nachdem Sie Ihre Änderungen vorgenommen haben, scrollen Sie nach unten zum Abschnitt Änderungen übernehmen. Fügen Sie eine Commit-Nachricht und optional eine Beschreibung hinzu und klicken Sie dann auf Änderungen übernehmen. Die Option „Änderungen übernehmen“ wird verwendet, um Ihre Änderungen im Repository zu speichern. Gehen Sie zurück zum Startbildschirm, indem Sie auf den Link mit dem Namen des Repositorys klicken. Beachten Sie, dass die Readme-Datei aktualisiert wurde, und überprüfen Sie Ihre Änderungen. Lassen Sie uns lernen, wie Sie mit dem integrierten Web-Editor von GitHub, der im Browser ausgeführt wird, eine neue Datei erstellen. Klicken Sie auf Datei hinzufügen und dann auf Neue Datei erstellen, um die neue Datei zu erstellen. Um eine Python-Datei namens firstpython.py zu erstellen. Geben Sie zunächst den Dateinamen an. Fügen Sie als Nächstes einen Kommentar hinzu, der Ihren Code beschreibt, und fügen Sie dann den Code hinzu. Sobald Sie fertig sind, übertragen Sie die Änderung in das Repository. Sie können sehen, dass Ihre Datei jetzt zum Repository hinzugefügt wurde und die Repository-Liste zeigt, wann die Datei hinzugefügt oder geändert wurde. Wenn Sie die Datei ändern müssen, können Sie sie erneut bearbeiten. Klicken Sie auf den Dateinamen und dann auf das Stiftsymbol, nehmen Sie Ihre Änderungen vor und übernehmen Sie die Änderungen. Sie können auch eine Datei von Ihrem lokalen System in das Repository hochladen. Klicken Sie auf dem Startbildschirm des Repositorys auf Datei hinzufügen und wählen Sie die Option Dateien hochladen. Klicken Sie auf Ihre Dateien auswählen und wählen Sie die Dateien aus, die Sie von Ihrem lokalen System hochladen möchten. Der Datei-Upload-Vorgang kann je nachdem, was Sie hochladen, eine kurze Zeit in Anspruch nehmen. Sobald der Upload der Dateien abgeschlossen ist, klicken Sie auf Änderungen übernehmen. Das Repository spiegelt jetzt die Dateien wider, die hochgeladen wurden. In diesem Video haben Sie gelernt, wie Sie mithilfe der Weboberfläche ein Repository erstellen, Dateien bearbeiten und Änderungen übernehmen.
